# ZOLTAN Travel Destiny Oracle

Modulo 41 de Gesture Lab.

## Que es

Skin ZOLTAN para turismo, hoteles, destinos y experiencias locales. El usuario piensa una ruta y ZOLTAN la revela con una recompensa accionable.

## Capas

- Hidden Engine: `ZoltanSelectionEngine`.
- Experience Skin: destino/ruta premium.
- Reward Layer: `route` o `generic_cta`.
- Host Voice: Route Oracle, inspirador y calido.

## Privacidad

No requiere camara. No usa backend. No hay QR falso ni datos personales.

## QA

- Abrir ruta.
- Completar con botones.
- Completar con zonas izquierda/derecha.
- Confirmar reveal.
- Confirmar reward.
- Descargar PNG.
- Ejecutar `window.__ZOLTAN_TRAVEL_QA__()`.
