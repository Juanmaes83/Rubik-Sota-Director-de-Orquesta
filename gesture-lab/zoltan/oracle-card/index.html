Cómo funciona
  1. Invoca el oráculo con botón, mano abierta o toque.
  2. Elige una carta con touch, mouse o cursor gestual.
  3. Mantén 0.8s sobre la carta para confirmar.
  4. Recibe reveal, CTA y recuerdo descargable.
