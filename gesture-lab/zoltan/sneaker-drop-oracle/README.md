# ZOLTAN Sneaker Drop Oracle

Modulo 40 de Gesture Lab.

## Que es

Skin ZOLTAN para retail hype y sneakers. El usuario piensa una sneaker, responde SI/NO y desbloquea un pase, wishlist, coupon o CTA local.

## Capas

- Hidden Engine: `ZoltanSelectionEngine`.
- Experience Skin: sneaker drop premium.
- Reward Layer: `drop_access`, `product`, `coupon` u `outfit`.
- Host Voice: Drop Oracle, urbano premium y complice.

## Privacidad

No requiere camara. No usa backend. No hay QR falso. No se piden datos personales.

## QA

- Abrir ruta.
- Completar con botones.
- Completar con zonas izquierda/derecha.
- Confirmar reveal.
- Confirmar reward.
- Descargar PNG.
- Ejecutar `window.__ZOLTAN_SNEAKER_QA__()`.
