# Air Draw

Modulo R&D del Gesture Gaming Pack para dibujo por gestos.

## Estado

- Tipo: server app / R&D.
- Estado: backend requerido.
- Repo: https://github.com/Juanmaes83/Air-Draw
- Demo publica GitHub Pages: no disponible.

## Motivo

Air Draw requiere Flask, SocketIO, MediaPipe y OpenCV. Por eso no debe copiarse dentro del core de Rubik como si fuera una pagina estatica.

## Integracion actual

Rubik contiene solo launcher estatico y documentacion de estado.
