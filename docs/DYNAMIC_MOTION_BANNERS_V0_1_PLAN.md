Dynamic Motion Banners v0.1 - Plan
==================================

Purpose
-------

Convert the raw BANDEROLAS-DINAMICAS WebGL experiment into a premium commercial module for Rubik Sota Director de Orquesta.

Strategic role
--------------

This is not a separate product.

It is a visual module inside Rubik Sota Director de Orquesta.

It supports the common engine:

Data -> Assets -> Preview -> QR/Landing -> Outreach -> Tracking -> CRM

Target use cases
----------------

- fashion campaign banners;
- interactive retail windows;
- storefront promotions;
- event posters;
- real estate campaign signs;
- food/restaurant promotion cards;
- hotel campaign previews;
- QR-based outreach pieces;
- premium digital signage.

What to preserve from the raw source
------------------------------------

- WebGL cloth/paper physics;
- Canvas 2D dynamic texture generation;
- image upload;
- video upload;
- PNG export;
- touch interaction;
- mouse interaction;
- physical motion feeling.
