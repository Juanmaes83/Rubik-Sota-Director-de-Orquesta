What it is
----------

A standalone HTML/WebGL experiment that creates an interactive dynamic banner using:

- Canvas 2D texture generation;
- WebGL rendering;
- custom shaders;
- Verlet cloth/paper physics;
- mouse interaction;
- touch interaction;
- image upload;
- video upload;
- PNG export.
