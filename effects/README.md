Effects Library
===============

Purpose
-------

Store reusable visual effects for Rubik Sota Director de Orquesta.

Effects can come from:

- internal experiments;
- GitHub repositories;
- CodePen;
- GSAP;
- CSS motion;
- WebGL;
- Canvas;
- particle systems;
- transitions;
- showroom interactions;
- storefront interactions.

Required lifecycle
------------------

raw-source
-> cleaned
-> production-ready
-> module integration

Rules
-----

- Raw sources are not production.
- Never publish raw sources directly.
- Every effect needs origin.
- Every effect needs permission/licensing status.
- Every effect needs mobile QA before production.
- Every effect needs performance QA before production.
- Every effect needs fallback before production.
